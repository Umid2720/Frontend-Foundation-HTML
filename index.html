Frontend Foundation: HTML

Bu repository HTML asoslarini o‘rganish uchun yaratilgan. Har bir darsda nazariya, amaliy topshiriqlar va misollar mavjud.

Darslar

1-dars — HTML bilan tanishuv

HTML nima, teg tushunchasi va oddiy sahifa yaratish.

2-dars — Text formatting

p, strong, em, br teglari bilan matnni formatlash.

3-dars — Headings

h1 dan h6 gacha bo‘lgan sarlavhalar va sahifa strukturasini tuzish.

Maqsad

Ushbu kurs HTML ni mutlaqo boshlang‘ich darajadan boshlab o‘rgatish uchun yozilgan. Darslar sodda va tushunarli tarzda berilgan.
